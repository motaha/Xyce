Windows users should execute the  runxyce.bat  batch file located in the 
bin  directory from the command line. 

    runxyce.bat [options] <netlist filename>



For help on using Xyce options, from the command line type:

    runxyce.bat -h


Complete documentation is provided in the  doc  directory.
